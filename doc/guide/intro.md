# 快速入门
新手快速上手指南（5分钟入门）

## 游戏核心目标
在战斗中击败敌方所有单位，同时保护我方角色不全部倒下。
## 核心战斗循环（每回合）

```
准备阶段 → 战斗阶段 → 下一回合
```

1. 准备阶段（你的规划时间）

- 投掷速度骰子 → 决定本回合行动顺序
- 抽1张牌，恢复1点COST → 资源补充
- 提升情感等级（如果情感积满）→ 强化角色
- 给速度骰子装备卡牌 → 规划行动

2. 战斗阶段（自动执行）

按照速度从高到低依次执行：

1. 群体攻击骰子
2. 群体治疗骰子
3. 单个攻击/防御/治疗骰子

> [!tip] 关键提示：速度高的骰子可以“截停”速度低的敌方骰子，强制进入拼点！

## 你必须知道的三个核心机制
### 机制一：拼点（战斗的核心）

· 触发：当你攻击的目标速度比你低时，强制进入拼点
· 过程：双方各掷骰子比大小
· 结果：
  · 赢：正常造成伤害
  · 输：攻击被抵消，骰子被破坏
· 简单记法：高速打低速 = 强制拼点；低速打高速 = 单方面攻击。

### 机制二：情感等级（战斗的关键）
**每次提升时都会提升 1 点 COST 上限并回满 COST。**
不仅如此，提升到一定程度时，你还可以**获得更多的速度骰子**，**更快速**以及**更强大的实力**。


### 机制三：理智系统（风险控制）

· 范围：-50 到 +50
· 影响：理智越高，掷骰数值越容易偏高
· 关键阈值：
  · 低于-30且队友一半也低于-30 → 士气低落
  · 达到-50 → 可能触发恐慌或情绪侵蚀

## 新手第一局建议

第一步：了解你的角色

1. 查看角色的被动
2. 查看关键技能

第二步：首回合操作

1. 装备攻击卡到高速骰子（截停敌人）
2. 装备防御卡到低速骰子（保护自己）
3. 保留1-2点COST备用

第三步：战斗中的决策重点

1. 理智监控：注意角色的理智
2. **情感等级：尽早提升，解锁更多骰子和COST**

📈 进阶技巧（打好基础后尝试）

1. 情绪克制：利用情绪关系获得伤害加成
2. 地形利用：让高地形适性角色进入掩体
3. 随机应变：若拼点不利可以只保留 1 颗较低速度的骰子拼点，其余单方面攻击。

🆘 遇到困难时检查

- 总是“卡费”？ 检查角色情感等级是否提升缓慢
· ❓ 总是拼点失败 → 检查理智是否过低
· ❓ 伤害太低 → 检查伤害抗性，是否使用了被抵抗的攻击类型
· ❓ 很快战败 → 检查是否没使用防御骰子
· ❓ 无法使用技能 → 检查COST是否足够

🎮 你的第一场战斗（模拟流程）

```
回合1：
- 准备阶段：投掷速度骰子（获得速度5、2）（假设你有速战速决）
- 装备：速度5（攻击卡）、速度2（防御卡）
- 战斗阶段：
  速度5：截停敌方速度4，拼点胜利，造成伤害
  速度2：防御状态（如果被攻击则自动防御）
- 结果：敌人1受伤，我方安全
```

记住：这个游戏的核心是“规划”而不是“反应”
在准备阶段做好决策，战斗阶段就会自动执行。多尝试几局，你就能掌握节奏！
