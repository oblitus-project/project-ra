# Project RA 文档导航

本文对 `doc/` 下各文档与目录进行定位说明，帮助新成员快速找到所需信息。

---

## `doc/build/` — 构建规格

游戏设计蓝图，定义实体数值与规则。面向开发者。

### `build/` 根目录文档

| 文件 | 定位 |
|------|------|
| `BOSS.md` | 头目角色构建（以 Binah 为例） |
| `Character.md` | 可玩角色构建模板/实例 |
| `Event.md` | 特殊事件（联合行动）存根 |
| `ExCharacter.md` | 额外角色：其他时间线的角色 |
| `Example.md` | 角色创建母版（模板） |
| `SlayCharacter.md` | 爬塔模式角色构建 |
| `SpCharacter.md` | 特殊 NPC/剧情战斗角色 |
| `Stage.md` | 关卡设计模板 |
| `Unit.md` | 普通单位/敌人构建模板 |

### `build/card/` — 卡牌定义

| 文件 | 定位 |
|------|------|
| `iron.md` | 铁血战士 的卡牌 |
| `upg-iron.md` | 上述卡牌的升级版（+） |
| `curse.md` | 诅咒卡牌 |
| `status.md` | 状态卡牌 |

### `build/keyword/` — 关键词体系

| 文件 | 定位 |
|------|------|
| `index.md` | 索引与分类映射 |
| `mechanism.md` | 核心机制（基础状态） |
| `effect.md` | 状态效果词典 |
| `emotion.md` | 情绪系统定义 |

---

## `doc/guide/` — 玩家指南

面向玩家的说明文档，帮助理解规则与上手游戏。

| 文件 | 定位 |
|------|------|
| `intro.md` | 快速入门教程（三场战斗教学） |
| `faq.md` | 进阶 FAQ（拼点/骰子/守备等） |
| `keyword.md` | 旧版关键词重定向/映射 |
| `library.md` | 术语库 v2.0（核心系统参考手册） |
| `affected.md` | 特效触发时机与条件 |
| `emotion.png` / `emotion.jpg` / `emotion.canvas` | 情绪系统可视化图表 |
